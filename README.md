# 🤖 Uptime Robot Telegram Bot

Bot de Telegram para monitorear tus webs usando Uptime Robot API.

## ✨ Características
- 📊 Ver estado de todas tus webs
- ➕ Añadir nuevas webs para monitorear
- ❌ Eliminar monitores existentes
- ⚙️ Configuración de alertas
- 🎨 Interfaz con botones interactivos

## 🚀 Despliegue en Render

1. Fork este repositorio
2. Conecta tu repo en Render
3. Configura las variables de entorno:
   - `TELEGRAM_TOKEN`
   - `UPTIME_API_KEY`
4. ¡Listo!

## 📱 Comandos
- `/start` - Menú principal

## 🛠 Tecnologías
- Python 3.9+
- python-telegram-bot v20
- Uptime Robot API v2
